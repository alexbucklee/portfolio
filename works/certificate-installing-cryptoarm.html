Установка сертификата с носителя

Проект в фигме

Доработка мастера установки сертификата подписи с внешнего носителя в приложении КриптоАРМ Mobile.

07/2024

  • UX/UI
  • Иллюстрации
  • Прототипирование

КриптоАРМ Mobile — мобильное приложение для электронной подписи и шифрования документов. Для создания электронной подписи в приложение необходимо добавить сертификат подписи. Сертификат подписи обычно хранится на внешнем защищенном носителе.

Задача

Помочь пользователю в установке сертификата с внешнего носителя, упростить процедуру установки и сделать ее более наглядной.

Предварительная работа

  • Неинтуитивный интерфейс: без ознакомления с документацией пользователю было непонятно, как именно установить сертификат с носителя в приложении.
  • Установка с USB-носителя: для установки сертификата с носителя нужно было подключить внешний носитель по usb (или lightning) порту к телефону. Носитель можно было подключить в любом разделе приложения, и сертификат автоматически устанавливался. При этом приложение не сообщало пользователю о том, что сертификат установлен.
  • Установка с NFC-карты: помимо USB-носителей с сертификатами существуют NFC-карты. Для установки сертификата с NFC-карты пользователю требовалось совершить другой сценарий, что создавало путаницу.

Результаты

Новый мастер по установке сертификата с носителя

Спроектировал объединенный мастер, который позволял пользователю устанавливать сертификат как с USB-носителя, так и с карты-носителя.

После установки сертификата система информирует пользователя о том, что он установлен.

Lottie-анимации

Для наглядности разработал lottie-анимации, которые демонстрировали подключение носителя к смартфону.
