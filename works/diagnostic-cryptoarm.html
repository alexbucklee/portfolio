Диагностика приложения

Проект в фигме

Новый функционал по настройке приложения в КриптоАРМ Mobile.

06/2024

  • UX/UI
  • Прототипирование

КриптоАРМ Mobile — мобильное приложение для электронной подписи и шифрования документов. Для того, чтобы выполнять криптографические операции в приложении, пользователю необходимо предварительно его настроить: установить лицензию, добавить сертификат подписи.

Задача

  • Улучшить пользовательский опыт: сделать процесс настройки приложения более прозрачным и понятным для пользователей. Особенно для тех, кто не знаком с тонкостями криптографической защиты информации.
  • Снизить нагрузку на техподдержку: уменьшить количество обращений в службу техподдержки.

Предварительная работа

  • Кейсы Выписал основные кейсы использования приложения.
  • Компоненты настройки Выявил компоненты приложения, которые необходимо настроить при первом запуске, в зависимости от кейса.
  • Поиск решения Для решения задачи рассматривалось два основных направления: пошаговый мастер настройки и чек-лист с настройками.

Результаты

Чек-лист для проверки рабочего места

В разделе Настройки появился блок для проверки рабочего места. В этом блоке отображаются компоненты системы, которые нужно настроить, а также их статусы. Статусы информируют пользователя о том, какие компоненты готовы к работе, а для каких требуется настройка.

Перейдя в компонент проверки, пользователь может совершить необходимое действие по настройке этого компонента.
